# ESP32 IR Remote Control


Dieses Projekt soll eine universelle Infrarot-Fernbedienung auf Basis eines ESP32 werden.

Ziel ist es, Infrarotsignale beliebiger Fernbedienungen aufzunehmen, zu speichern und später wiederzugeben. Die Bedienung soll über eine Weboberfläche erfolgen, die direkt auf dem ESP32 läuft. Nutzer sollen eigene virtuelle Fernbedienungen erstellen und gespeicherte Signale einzelnen Schaltflächen zuordnen können.


## Geplante Funktionen

* WLAN-Anbindung des ESP32
* Integrierter Webserver
* Aufnehmen von IR-Signalen
* Speichern und Verwalten von Signalen
* Wiedergeben gespeicherter Signale
* erstellbare Fernbedienungs-Buttons in der Weboberfläche
